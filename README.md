# Wealthfolio CSV Importer Scripts

Python tools to transform Schwab and Fidelity transaction exports into a single Wealthfolio-compatible CSV.

## ✨ Features

- 🔄 Combine multiple Fidelity/HSA files
- 🔍 Smart categorization (dividends, fees, deposits)
- 📅 Chronological sorting & deduplication
- 💡 Handles edge cases (cash in lieu, FX adjustments)
